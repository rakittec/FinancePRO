## 🌟 **Kenapa PersonalFinancial?**

Bukan sekadar aplikasi catatan keuangan. PersonalFinancial adalah **asisten keuangan pribadi** yang dirancang dengan sentuhan desain modern dan pengalaman pengguna yang mulus. Semua data tinggal di perangkat Anda — **tanpa internet, tanpa cloud, tanpa biaya**.

> ✨ *"Aplikasi ini adalah wujud dari keyakinan bahwa mengelola uang harus terasa menyenangkan, bukan membebani."*

---

## 🧩 **Fitur Unggulan**

| Kategori | Fitur |
|----------|-------|
| 📊 **Dashboard Cerdas** | Saldo total, grafik tren 30 hari, statistik bulanan, dan progress goals dalam satu layar |
| 💸 **Transaksi Lengkap** | Tambah/edit/hapus, filter (jenis, rekening, bulan/tahun/tanggal), pencarian, paginasi, upload bukti, ID transfer |
| 🏦 **Rekening Fleksibel** | Buat rekening dengan warna kustom, saldo otomatis update, edit/hapus kapan saja |
| 🎯 **Goals & Reward** | Tetapkan target, setor harian, pantau progress, raih reward, lihat riwayat setoran |
| 🌙 **Dark Mode** | Nyaman dipakai malam hari, preferensi tersimpan otomatis |
| 📁 **Backup & Restore** | Export data ke Excel (.xlsx), import dari Excel kapan saja |
| ⚙️ **Personalisasi** | Ubah nama aplikasi, reset data total (dengan konfirmasi ganda) |
